## 三、技术架构总览

```
┌─────────────────────────────────────────────┐
│                  浏览器                       │
│         Bootstrap 5 + Leaflet.js             │
└──────────────────┬──────────────────────────┘
                   │ HTTP
┌──────────────────▼──────────────────────────┐
│              Nginx (端口80)                   │
│         反向代理 + 静态资源服务                 │
└──────────────────┬──────────────────────────┘
                   │ Unix Socket
┌──────────────────▼──────────────────────────┐
│          Gunicorn (WSGI 服务器)               │
│           9 worker 进程                       │
└──────────────────┬──────────────────────────┘
                   │
┌──────────────────▼──────────────────────────┐
│            Flask 应用                         │
│  ┌─────────┬──────────┬──────────┬────────┐ │
│  │  社区   │  众筹    │  手账    │  驿站  │ │
│  ├─────────┴──────────┴──────────┴────────┤ │
│  │     Auth  │  User Center  │  Admin     │ │
│  ├────────────────────────────────────────┤ │
│  │  Flask-Babel (i18n) │ Flask-Login      │ │
│  ├────────────────────────────────────────┤ │
│  │         SQLAlchemy ORM                 │ │
│  └────────────────┬───────────────────────┘ │
└───────────────────┬─────────────────────────┘
                    │
┌───────────────────▼─────────────────────────┐
│            SQLite 数据库                      │
│         12 张表，含 RBAC 权限                  │
└─────────────────────────────────────────────┘
```
